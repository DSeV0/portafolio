PelixPlus

Paginá creada con fines de practica y aprendizaje. Utilizando principalmente plugins de wordpress como Custom Post Type, se generan las entradas para cada pelicula y se modifica el contenido a la vista, cambiando entre el menú con todas las peliculas y una vista especifica de cada una, con información sobre ella, fecha de estreno, un personaje favorito y uno destacado. El contenido de cada pelicula puede ser modificado dentro del sitio mediante un formulario de edición. Tambien se pueden agregar nuevas peliculas estando dentro del sitio.
